Things to do...

1.    Download SSH Secure Shell Client & install it to your laptop (if it is not already installed to your laptop)
      

2.    Download ClustalX  &  GeneDoc and install them
      

3.     Download & install Java from Java download page.


4.    Download Python 2.7.1 (32 bit Laptop) or Python 2.7.1 (64 bit Laptop) for Windows platform. If you have Mac OS, then download according to your laptop configuration.


5.     Download Biopython 1.57 (32 bit Laptop) or  Biopython 1.57 (64 bit Laptop)  for Windows platform. If you have Mac OS/Linux, then download according to your laptop configuration.
