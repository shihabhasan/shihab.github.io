== Description ==

fPsychology is a fully responsive WordPress Theme, a perfect choice for a website or blog about art, mastery,
artistic, art, artistical, etc. Features include Multi-Level Drop-Down Header Menu, W3C Markup Validated,
4 Homepage Widget Areas, 3 Footer Column Widget Areas, 1 Sidebar Widget Area, Admin Options (Social Sites,
Slider Images and Content, Header Logo and Footer Copyright Text), Search Engine Optimized (SEO),
Multiple Browser Compatible (IE9, IE10, IE11, Chrome, Firefox, Opera, and Safari), FontAwesome Integration,
and Translation Ready.

== Installation ==

Manual installation:

1. Upload the theme folder to the /wp-content/themes/ directory

Installation using 'Add New Theme'

1. From your Admin UI (Dashboard), use the menu to select Themes -> Add New
2. Search for theme name
3. Click the 'Install' button to open the theme's repository listing
4. Click the 'Install' button

Activiation and Use

1. Activate the Theme through the 'Themes' menu in WordPress
2. See Appearance -> Theme Options to change theme specific options
